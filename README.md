AFAIR UI 2013 Website

## This is a Website for AFAIR UI 2013 Event.
List of plugins, contents and stuff i used for this website:

#### What i use?
* Elementary OS Luna - [Elementary OS](http://www.elementaryos.org/ "Elementary OS")

* VIM
* Github of course
* My Brain
* URXVT
